\subsection{Detailed Description}
A surface representing a pane of widgets. 

Panes have a frame and a semi-\/transparent background color. This class is meant to be subclassed for use in {\ttfamily \hyperlink{class_u_i_1_1_window_contents}{Window\-Contents}} objects.
