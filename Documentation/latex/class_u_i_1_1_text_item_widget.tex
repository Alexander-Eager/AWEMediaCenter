\subsection{Detailed Description}
Displays text in a specific font. 

Unlike {\ttfamily \hyperlink{class_u_i_1_1_text_pane}{Text\-Pane}}, no background is drawn, so it is safe to use in lists and in {\ttfamily \hyperlink{class_u_i_1_1_pane}{Pane}}s.

In addition, this class signals mouse click events
