\subsection{Detailed Description}
An abstract, scrollable collection of item widgets. 

Provides convenience functions for displaying a collection of {\ttfamily \hyperlink{class_u_i_1_1_item_widget}{Item\-Widget}}s and receiving information on their state.

When using this class (or a sub-\/class), you should never need to call any of {\ttfamily Q\-Scroll\-Area}'s functions.
