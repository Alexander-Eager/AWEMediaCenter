\subsection{Detailed Description}
Holds the global settings for \hyperlink{class_a_w_e_1_1_a_w_e_m_c}{A\-W\-E\-M\-C}. 

\begin{DoxyRefDesc}{Todo}
\item[\hyperlink{todo__todo000008}{Todo}]Skins 

\hyperlink{namespace_u_i}{U\-I} delegate (with user prompting)\end{DoxyRefDesc}


Functions for pretty much anything you want to know about the prefernces for \hyperlink{class_a_w_e_1_1_a_w_e_m_c}{A\-W\-E\-M\-C} are here. Every plugin extending \hyperlink{class_a_w_e_1_1_a_w_e_m_c}{A\-W\-E\-M\-C} has its own configuration file, which is held in here via sub-\/configuration objects.

You shouldn't need to instantiate this class on your own; most objects are passed a copy of the settings object during creation or through some other means.
