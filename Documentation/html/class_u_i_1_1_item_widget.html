Detailed Description

Defines an abstract item widget for lists.

It behaves in a way similar to QListWidgetItem, but unfortunately QListWidget has an opaque background and does not work well with AWEMC. I took this opportunity to make a set of similar widgets that do not have this problem.

Subclasses need to define:

  • getItemType(): the type of ItemWidget (for casting purposes).
  • paintEvent(): to paint the widget.
  • The minimum size of the widget (so that it shows up).

virtual bool UI::ItemWidget::fixSizeToFitIn ( QSize  size)
pure virtual

Fix this item's size to fit in the given size.

If size == QSize(-1, -1), the item should not fix its size.

This is mostly used by ItemCollectionWidgets to make sure that the item does not get swallowed up by the spacers used to align the items.
