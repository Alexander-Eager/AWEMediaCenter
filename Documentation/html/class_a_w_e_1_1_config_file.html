Detailed Description

Holds the information for a configuration file in an abstract way.

The configuration files used by AWEMC are all JSON files.

virtual QVariant AWE::ConfigFile::getMember ( QString  path) const
virtual

Get a data member.

path is formatted so that delving down a member uses a ".". So if I have: 

{
    "a": {
        "b": [
            "Hello World"
        ]           
    }
}

The pathing to use to get to "Hello World" is "a.b.0".

Parameters
[in]pathThe path to the data member to retrieve.
Returns
The desired data member, or a null QVariant if the member does not exist.
virtual QVariant AWE::ConfigFile::getMember ( QVariantList  path) const

Get a data member.

path is a list of values to follow to reach the desired data member. So if I have: 

{
    "a": {
        "b": [
            "Hello World"
        ]           
    }
}

The list to use to get to "Hello World" is { "a", "b", 0 }.
